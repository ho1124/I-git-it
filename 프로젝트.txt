이클립스에서 파일 넣는법
1.git init
2.git add 넣고 싶은 파일명
3.git commit -m ""
4.커밋 확인 git log
5.add 확인 git status
6.git branch -M main(이미있는 것은 브렌치명 다르게 브랜치명 중복 불가능)
7.git remote add origin https://github.com/ho1124/i-git-it.git
8.git push -u origin main
